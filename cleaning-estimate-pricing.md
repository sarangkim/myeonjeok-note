# 청소 견적 계산기 초기 단가 메모

이 문서는 면적노트의 첫 견적 계산기에 넣은 초기 단가와 보정식을 정리한 메모다. 실제 계약 데이터가 쌓이면 이 값을 앱의 `calculateEstimate()` 함수에서 조정한다.

## 참고한 공개 가격대

- 사무실 정기청소: 쓱싹 기준 30평 이하 주 1회 월 20만원부터, 주 2회 35만원부터, 주 3회 45만원부터. 31~50평은 주 1회 30만원부터, 주 2회 50만원부터, 주 3회 65만원부터.
- 사업장/사무실 1회성 청소: 아정당 안내 기준 사업장 청소 평당 15,000~30,000원.
- 사무실·상가 및 준공청소: 청광 안내 기준 사무실·상가 평당 10,000~20,000원, 준공·건축 청소 평당 15,000~25,000원.
- 아파트 입주청소: 아정당 기준 평당 12,000~15,000원, AARO 기준 2026년 3월 평당 10,000~15,500원, 카더라 2026 가격표 기준 20평대 25~35만원, 30평대 35~50만원.
- 빌라 계단청소: 굿페이스클린 기준 4~5층 월 1회 35,000원, 미소크린 기준 5층 8~10세대 월 4회 6~7만원.

## 앱에 넣은 초기 산식

공통 보정:

```text
예상가 = (기본가 + 옵션가) * 오염도계수 * 집기계수 * 야간주말계수
표시범위 = 예상가의 90% ~ 115%
```

오염도 계수:

```text
낮음 0.92 / 보통 1.00 / 높음 1.18 / 매우 높음 1.38
```

집기 계수:

```text
비어 있음 1.00 / 일부 있음 1.08 / 많음 1.20
```

청소 종류별 기본가:

```text
사무실 정기청소 = 면적 구간별 월 기본가 * 방문횟수계수 + 화장실 8,000원/개 + 방 4,000원/개
병원 정기청소 = 사무실 정기청소 * 1.35 + 화장실 12,000원/개 + 진료실 8,000원/개
학원 정기청소 = 사무실 정기청소 * 1.18 + 화장실 9,000원/개 + 강의실 6,000원/개
사무실 입주청소 = max(250,000원, 평수 * 18,000원) + 화장실/방 보정
준공청소 = max(350,000원, 평수 * 20,000원) + 방/폐기물 보정
대청소 = max(220,000원, 평수 * 13,000원) + 화장실/방 보정
아파트 입주청소 = max(150,000원, 평수 * 14,000원) + 화장실 22,000원/개
빌라 계단청소 = max(35,000원, 층수 * 12,000원 + 세대수 * 1,200원) * 월방문횟수/4
```

옵션:

```text
창틀·베란다 중점 = max(30,000원, 평수 * 1,200원)
바닥 왁스·코팅 = max(50,000원, 평수 * 3,500원)
폐기물·보양재 정리 = max(30,000원, 평수 * 2,000원)
야간·주말 = 15% 가산
```

## 주의

이 계산기는 확정 견적이 아니라 초기 상담용 예상 범위다. 실제 금액은 사진, 현장 방문, 주차·엘리베이터, 물 사용 가능 여부, 외창/고소작업, 폐기물, 지역 인건비에 따라 달라진다.
